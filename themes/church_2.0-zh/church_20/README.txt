Revolution Church WordPress Theme
http://www.revolutiontwo.com/themes/church

INSTALL: 
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to Admin WordPress and select Appearance.
3. Select Revolution Church WordPress theme.
4. Download the latest version of the Featured Content Gallery plugin (http://www.revolutiontwo.com/plugins/featured-content-gallery.htm)
5. Upload the Featured Content Gallery plugin via FTP to your wp-content/plugins directory, and follow the directions from the plugin page to implement it.

If you are looking for theme support, please visit http://www.revolutiontwo.com/support
